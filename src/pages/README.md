# GigFlow – Smart Leads Dashboard

A full-stack tracking pipeline built with React, Node.js, TypeScript, Express, and MongoDB.

## Features
- **Dynamic Search & Filtering:** Filter leads instantly by status, source, or text query.
- **Server-Side Pagination:** Efficiently loads data chunks from MongoDB.
- **CSV Exporter:** Download current data arrays into a local spreadsheet report.
- **Robust Types:** Self-contained TypeScript configurations preventing data leaks.

## Local Setup Instructions

### Backend
1. Navigate to `/backend`
2. Run `npm install`
3. Seed the database with sample entries: `npm run seed`
4. Start development server: `npm run dev`

### Frontend
1. Navigate to `/frontend`
2. Run `npm install`
3. Start frontend workspace: `npm run dev`
